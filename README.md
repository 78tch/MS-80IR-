# MS-80IR-
## 1.概要
1. 使えるモデルは３つ（エフェクト、アンプ、スタジオアンビエンス）
2. スタジオアンビエンスの代わりとしてマルチレイヤIRも使える。その場合、アンプのキャビシュミはバイパスされる。
3. IRは、44.1～192kHzのWAVが使えるが、ZOOMのHPから「マルチレイヤIR」がDLできる。
4.
